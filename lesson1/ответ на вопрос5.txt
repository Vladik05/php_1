Используя только две переменные, поменяйте их значение местами. Например, если a = 1, b
= 2, надо, чтобы получилось: a = 2b = 1, . Дополнительные переменные использовать нельзя.

ответ:

    $a = 1;
    $b = 2;
    $a = $a + $b;
    $b = $b - $a;
    $b = -$b;
    $a = $a - $b;
    echo $a.'- это a <br>';
    echo $b.'- это b';
